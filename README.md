User Management System

A simple and scalable User Management System built with PHP and MySQL.
This project handles user authentication, role-based access control, and basic user CRUD operations. Ideal for learning or as a starter backend for web apps.

Features

User registration & login

Secure authentication (password hashing)

Role-based access (Admin / User)

Create, read, update, delete (CRUD) users

Session-based login system

Clean and structured PHP code

MySQL database integration

Tech Stack

Backend: PHP (Core PHP / MVC-style structure)

Database: MySQL

Frontend: HTML, CSS (Bootstrap optional)

Server: Apache (XAMPP / WAMP / Laragon)
